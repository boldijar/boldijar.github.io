Livetickets

Android app for people that work at festivals.

With this app you can scan tickets at a festival, you can do this also offline, because the app will take care of the ticket sync when the internet connection comes back. You can scan the ticket by it's QR code or manually by search. You can also see stats about the current event and much more.

  • Date: September 2017
  • Client: Livetickets.ro
  • Technologies used: Dagger 2, Retrofit, Okhttp, Butterknife...

Colorchat

Basic realtime chat application

In this application you can join a group, and simply chat with other participants. No message is persisted, so when you open the app again, you won't see any old message. You can choose a certain color, that the other participants will see. This app only uses Firebase Cloud Messaging.

  • Date: August 2017
  • Project type: personal project
  • Technologies: Firebase Cloud Messaging

Your way

Navigation app

This app was build at iTECH hackathon. Using this app you can see your current route, coloured by the speed you have. The backend was done in Javascript. This app has some extra features such as adding photos to custom locations, seeing where are the traffic jams and getting some statistical information about your driving. Won the 2nd place with this app.

  • Date: April 2017
  • Project type: hackathon app
  • Technologies used: Facebook SDK, Google Maps API
